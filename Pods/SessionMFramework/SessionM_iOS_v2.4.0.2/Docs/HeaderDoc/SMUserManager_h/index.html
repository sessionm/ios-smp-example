NS_SWIFT_NAME

Notifies observers that account data for the current user was updated. 

extern NSString *const SM_USER_MANAGER_DID_UPDATE_USER NS_SWIFT_NAME(
    updatedUserNotification);
Discussion

An SMPUser object containing information about the current user's account can be accessed from the notification's userInfo property with the SM_MANAGER_NOTIFICATION_DATA_KEY key, and an NSSet of the names of the user properties that were updated can be accessed with the kUserDeltas key.

NS_SWIFT_NAME(userRequestFailureNotification)

Notifies observers that a User API request failed. 

extern NSString *const SM_USER_MANAGER_REQUEST_DID_FAIL_NOTIFICATION NS_SWIFT_NAME(
    userRequestFailureNotification);
Discussion

An SMError object containing information about why the request failed can be accessed from the notification's userInfo property with the SM_MANAGER_NOTIFICATION_ERROR_KEY key.

NS_SWIFT_NAME(userValidationFailureNotification)

Notifies observers that a User API validation failed. 

extern NSString *const SM_USER_MANAGER_VALIDATION_DID_FAIL_NOTIFICATION NS_SWIFT_NAME(
    userValidationFailureNotification);
Discussion

An SMError object containing information about why the validation failed can be accessed from the notification's userInfo property with the SM_MANAGER_NOTIFICATION_ERROR_KEY key.
