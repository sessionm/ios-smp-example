Superclass:
NSObject

Introduction

This API manager contains methods for uploading receipt images and fetching uploaded receipt data.

groupIDsForIncompleteUploads

Returns identifiers for the current user's incomplete receipt uploads (uploads from other users will be removed). 

- (NSArray<NSString *> *)incompleteUploadGroupIDs NS_SWIFT_NAME(groupIDsForIncompleteUploads());
Return Value

NSArray of upload identifiers.

Discussion

An element of the returned array can be used as input for resumeIncompleteUploadForGroupID:completionHandler:progressHandler: or removeIncompleteUploadForGroupID:.
