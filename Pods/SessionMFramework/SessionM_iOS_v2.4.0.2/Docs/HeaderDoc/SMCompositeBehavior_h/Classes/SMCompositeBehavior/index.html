Introduction

Defines the data associated with a composite action item comprising other count and unique behaviors.


Methods

-goalsForGroupID:forGroupID:

Returns goals associated with the behavior that belong to the rule group with the specified identifier.

goalsForGroupID:forGroupID:

Returns goals associated with the behavior that belong to the rule group with the specified identifier. 

- (NSDictionary<NSString *, SMGoal *> *)goalsForGroupID:(NSString *)groupID NS_SWIFT_NAME(goals(
                forGroupID:));
Parameters
groupID

The identifier for the rule group whose goals will be returned.

Return Value

NSDictionary of mappings from behavior names to goals.
