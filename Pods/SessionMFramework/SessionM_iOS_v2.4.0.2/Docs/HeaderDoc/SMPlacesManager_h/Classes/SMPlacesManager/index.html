Introduction

This API manager contains methods for fetching and checking into sponsored venues.

Makes a request to update places with a limited number of venues available for check in inside the circle with the specified radius centered at the specified location. 

- (BOOL)fetchPlacesAroundLocation:(CLLocation *)location 
        withRadius:(float)radius limit:(int)limit adUnitID:(NSString * _Nullable)adUnitID 
        completionHandler:(didFetchPlaces)completionHandler NS_SWIFT_NAME(fetchPlaces(
                near:radius:limit:adUnitID:completionHandler:));
Parameters
location

The center of the circle in which places will be fetched.

radius

The radius in meters of the circle in which places will be fetched (optional).

limit

The maximum amount of places to fetch (optional).

adUnitID

Fetched places that do not have a bonus opportunity linking to the ad unit with the specified ID will be filtered out (optional - developers can setup ad units from the Campaigns module in the SessionM Platform portal).

completionHandler

The block to execute after the request is processed.

Return Value

BOOL indicating whether the request will be sent.
