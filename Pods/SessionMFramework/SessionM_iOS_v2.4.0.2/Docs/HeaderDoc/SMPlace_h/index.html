Constants
SMCheckinStatusNotCheckable

User cannot check into the venue.

SMCheckinStatusCheckable

User is within range of the venue.

SMCheckinStatusAlreadyCheckedInCheckable

User is within range of the venue, but cannot check in because they have recently checked in.

SMCheckinStatusAlreadyCheckedInTooFar

User is too far to check into the venue, and have recently checked in.

SMCheckinStatusBlockedCheckable

User is within range of the venue, but cannot check in because they have reached the checkin limit for the venue's brand or category.

SMCheckinStatusBlockedTooFar

User is too far to check into the venue, and has reached the checkin limit for the venue's brand or category.

SMCheckinStatusBlockedLimitHit

User has reached the daily checkin limit.

SMCheckinStatusTooFar

User is too far to check into the venue.
