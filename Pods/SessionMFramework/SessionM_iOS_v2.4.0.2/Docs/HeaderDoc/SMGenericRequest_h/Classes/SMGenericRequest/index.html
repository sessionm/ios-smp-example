SMGenericRequest

Superclass:
NSObject
Declared In:
SMGenericRequest.h

Introduction

Defines the data associated with a generic SessionM Platform HTTP request.

Discussion

The following tokens can be used for formatting:

  • ':host:' - the SessionM Platform API host URL

  • ':api_key:' - the app's SessionM Platform API key
