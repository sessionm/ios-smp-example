NS_SWIFT_NAME(sentReferralsNotification)

Notifies observers that referrals were sent. 

extern NSString *const SM_REFERRALS_MANAGER_DID_SEND_REFERRALS_NOTIFICATION NS_SWIFT_NAME(
    sentReferralsNotification);
Discussion

An NSArray of successfully sent SMReferral objects can be accessed from the notification's userInfo property with the SM_MANAGER_NOTIFICATION_DATA_KEY key. If any referral requests were denied, then an NSArray of SMReferralError objects containing information about why a particular referral was denied can be accessed with the SM_REFERRALS_MANAGER_NOTIFICATION_ERRORS_KEY key - and a general SMError object can be accessed with the SM_MANAGER_NOTIFICATION_ERROR_KEY key.
