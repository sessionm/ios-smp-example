Superclass:

Introduction

This API manager contains methods for fetching promotional campaign data.


Methods

-executeActionForMessage:for:

Executes the action associated with the specified message.

-fetchFeedMessages

Makes a request to update feedMessages with promotional campaigns targeted to the current user's locale (default value is [ SessionM customLocale ] if set and [NSLocale currentLocale] otherwise).

-fetchFeedMessagesWithCompletionHandler:completionHandler:

Makes a request to update feedMessages with promotional campaigns targeted to the current user's locale (default value is [ SessionM customLocale ] if set and [NSLocale currentLocale] otherwise).

-fetchFeedMessagesWithLocale:completionHandler:for:completionHandler:

Makes a request to update feedMessages with a number of promotional campaigns targeted to the specified locale.

Makes a request to update feedMessages with promotional campaigns targeted to the specified locale.
