Introduction

This API manager contains methods for fetching promotional content data records.


Methods

-fetchContentsForPlacement:limit:page:completionHandler:havingPlacement:limit:page:completionHandler:

Makes a request to update contents with a limited number of content records that have the specified content feed placement value.

fetchContentsForPlacement:limit:page:completionHandler:havingPlacement:limit:page:completionHandler:

Makes a request to update contents with a limited number of content records that have the specified content feed placement value. 

- (BOOL)fetchContentsForPlacement:(NSString * _Nullable)placement 
        limit:(NSInteger)limit page:(NSInteger)page 
        completionHandler:(didFetchContents)completionHandler NS_SWIFT_NAME(fetchContents(
                havingPlacement:limit:page:completionHandler:));
Parameters
placement

Filters the fetched content records based on where the containing content feed is displayed (e.g. "front_page") (optional).

limit

The maximum amount of content records to fetch (optional).

page

limit * (page-1) most recent content records will be skipped for the fetch (optional).

completionHandler

The block to execute after the request is processed.

Return Value

BOOL indicating whether the request will be sent.
