Superclass:
NSObject
Declared In:

Introduction

This API manager contains methods for viewing web pages relating to a user's profile (e.g. updating a user's account data, resetting a user's password, etc.).


Methods

+instance

Singleton that interfaces with the SessionM Platform User Web Profile API.

-updateUserProfileInViewController:presentationHandler:dismissalHandler:in:presentationHandler:dismissalHandler:

Presents an instance of SFSafariViewController (the Safari app is opened instead on iOS 8 devices) that points to a web page where the user's account information can be viewed and updated.

Discussion
