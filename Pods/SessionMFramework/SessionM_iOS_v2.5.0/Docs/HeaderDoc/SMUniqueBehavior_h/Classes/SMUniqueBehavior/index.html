iOS SDK Documentation

SMUniqueBehavior

Superclass:
SMProgressBehavior
Declared In:
SMUniqueBehavior.h

Introduction

Defines the data associated with an action item associated with a campaign or incentive that tracks unique actions that the user has made.


Properties

achieved

Number of times the behavior has been completed.

completedUniques

Names of the events that the user has already logged.

currentCount

Number of unique events that have been logged towards completing the behavior.

data

Additional data associated with the behavior.

maxTimesRepeatable

Maximum number of times a user is allowed to complete the behavior.

points

Number of points awarded to the user for completing the behavior.

totalCount

Total number of unique events that need to be logged before the behavior is completed.
