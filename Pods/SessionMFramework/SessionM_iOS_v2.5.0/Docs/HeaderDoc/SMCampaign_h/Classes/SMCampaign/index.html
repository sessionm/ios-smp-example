SMCampaign

Superclass:
NSObject
Declared In:
SMCampaign.h

Introduction

Defines the data associated with a campaign message.

Discussion

Note: the developer can configure the following properties for each message through the SessionM Platform portal.


Properties

campaignID

Identifier of campaign associated with the message.

creativeID

Unique identifier for campaign creative.

endTime

Indicates when the campaign ends.

extras

Any data values that do not have an associated class property.

groupID

Identifier of ad group that contains the campaign.

messageID

Unique identifier for message.

name

Campaign name.

optInEndsAt

Indicates when opt-in for the campaign ends.

optInStartsAt

Indicates when opt-in for the campaign began.

permalink

Developer-defined string that can be used to reference the campaign.

startTime

Indicates when the campaign began.

type

Message type.

version

SessionM Platform Campaigns API version.
