SMInboxManager

Superclass:
NSObject
Declared In:
SMInboxManager.h

Introduction

This API manager contains methods for fetching and updating message data that can be organized into a user inbox.


Methods

-createInboxMessage:completionHandler:_:completionHandler:

Makes a request to create a message with the specified configuration for the current user.

-fetchInboxMessagesWithCompletionHandler:completionHandler:

Makes a request to update inboxMessages with the current user's inbox messages.

-updateInboxMessage:state:completionHandler:_:toState:completionHandler:

Makes a request to update the specified message with the specified state.

-updateInboxMessageID:state:completionHandler:havingID:toState:completionHandler:

Makes a request to update the message with the specified identifier with the specified state.

-updateInboxMessageIDs:state:completionHandler:havingIDs:toState:completionHandler:

Makes a request to update the messages with the specified identifiers with the specified state.
