Superclass:
NSObject
Declared In:

Introduction

Class used for building SMEventItem objects that represent the data associated with a single instance of a triggered event.


Methods

-build

Generates an event based on the builder's current configuration.

+builder

Returns an event item builder.

-setAttribute:forKey:value:forKey:
