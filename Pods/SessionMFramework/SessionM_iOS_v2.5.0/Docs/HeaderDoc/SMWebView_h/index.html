SMWebView

A WKWebView wrapper that can register objects to handle JSBridge messages and redirects initiated by the SessionM Platform.

Protocols

SMWebViewDelegate

Defines callbacks to be implemented by delegates of SMWebView instances.

extern NSString *const SM_WEB_VIEW_ERROR_NOTIFICATION NS_SWIFT_NAME(
    webViewErrorNotification);
Discussion

An SMError object containing information about why the JavaScript evaluation failed can be accessed from the notification's userInfo property with the kSMWebViewError key.

NS_SWIFT_NAME(webViewResultNotification)

Notifies observers of successful JSBridge handler evaluations. 

extern NSString *const SM_WEB_VIEW_RESULT_NOTIFICATION NS_SWIFT_NAME(
    webViewResultNotification);
Discussion

An id object containing data returned by the JavaScript evaluation can be accessed from the notification's userInfo property with the kSMWebViewData key.

Constants

kSMWebViewData

Used to access data returned by a successful JavaScript evaluation in a SM_WEB_VIEW_RESULT_NOTIFICATION object's userInfo property.

kSMWebViewError

Used to access an SMError object from a failed JavaScript evaluation in a SM_WEB_VIEW_ERROR_NOTIFICATION object's userInfo property.
