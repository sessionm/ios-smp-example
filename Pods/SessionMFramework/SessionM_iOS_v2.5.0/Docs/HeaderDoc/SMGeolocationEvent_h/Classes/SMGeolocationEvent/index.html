Introduction

Defines the data associated with a location-based event and its associated geofence.

Discussion

Location event properties can be configured from the SessionM Platform portal's Behaviors module.


Properties

delay

Specifies how long a user must remain in the geofence to trigger the location event when triggerType is set to SMGeolocationEventTriggerTypeDwell.

distance

Distance from [ SMLocationManager currentGeoLocation ] to the geofence center point.

eventName

Location event behavior name.

identifier

Unique identifier that is tied to the geo location entry.

latitude

Latitude for the geofence center point.

longitude

Longitude for the geofence center point.

metaData

Meta data associated with the location event.

radius

Geofence radius of effect (in meters).

triggerType

Specifies how the location event is triggered.

triggerTypeString

Corresponding string value for triggerType.
