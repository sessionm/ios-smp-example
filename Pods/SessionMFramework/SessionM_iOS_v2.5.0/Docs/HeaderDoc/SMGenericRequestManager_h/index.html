Discussion

An SMError object containing information about why the request failed can be accessed from the notification's userInfo property with the SM_MANAGER_NOTIFICATION_ERROR_KEY key.

NS_SWIFT_NAME(genericRequestSuccessNotification)

Notifies observers that an API request succeeded. 

extern NSString *const SM_GENERIC_REQUEST_MANAGER_DID_SUCCEED_NOTIFICATION NS_SWIFT_NAME(
    genericRequestSuccessNotification);
Discussion

An SMGenericResponse object containing information about the request response can be accessed from the notification's userInfo property with the SM_GENERIC_REQUEST_MANAGER_NOTIFICATION_RESPONSE_KEY key.
