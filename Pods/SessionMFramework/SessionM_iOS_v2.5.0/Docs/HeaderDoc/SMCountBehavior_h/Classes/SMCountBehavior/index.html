Superclass:

Introduction

Defines the data associated with a simple action item associated with a campaign or incentive.


Properties

achieved

Number of times the behavior has been completed.

consecutive

Requires that event must happen on consecutive days.

currentCount

Current number of events logged towards completing the behavior.

eventName

Name of the event associated with the behavior.

maxTimesRepeatable

Maximum number of times a user is allowed to complete the behavior.

maxTimesRepeatablePerPeriod

Maximum number of times a user is allowed to complete the behavior in the time interval specified by period.

minTimeBetweenEvents

Minimum amount of time a user is required to wait between attempts.

period

Defines the time interval (in seconds) that applies to maxTimesRepeatablePerPeriod.

points

Number of points awarded to the user for completing the behavior.

totalCount

Total number of events that need to be logged before the behavior is completed.
